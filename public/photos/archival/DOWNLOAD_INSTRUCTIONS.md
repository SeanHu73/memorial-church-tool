# Archival Photo Download Instructions

All images are freely usable (no known restrictions). Download each file manually
from the LOC page linked below and save it to this folder (`public/photos/archival/`)
using the filename shown. All captions verified directly from the LOC gallery or item page.

Full metadata for all files is in `docs/archival_photo_inventory.csv`.
